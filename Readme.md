## Speedy Routiny

This is my final project while attending The Iron Yard.  Speedy Routiny is an app created to help kids quickly complete morning & nighttime routines and (hopefully) have fun doing it.

### Tech Used

1. Backbone
2. jQuery
3. Underscore
4. Flipclock.js
5. Amcharts
6. SASS
7. Modernizr
8. Bourbon / Neat
9. Parse
